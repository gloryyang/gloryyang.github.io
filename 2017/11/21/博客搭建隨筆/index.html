博客搭建隨筆

Nov 21, 2017

安装Hexo

npm install hexo-cli -g
hexo init #初始化

hexo g # 生成静态文件，会在当前目录下生成一个新的叫做public的文件夹
hexo s # 启动本地web服务，可以在http://localhost:4000/ 查看

_config.yml配置文件的修改

安装主题

git clone https://github.com/wwwuxt/hexo-theme-futimpX.git themes/futimpX

启用主题

修改Hexo目录下的_config.yml配置文件中的theme属性，将其设置为futimpX

更新主题

cd themes/futimpX
git pull
hexo g
hexo s
现在打开http://localhost:4000/ ，会看到我们已经应用了一个新的主题。

部署Hexo到Github Pages

将hexo生成的静态网站（public文件夾），提交(git commit)到github上。

需要提前安装一个扩展：

npm install hexo-deployer-git –save

在配置文件_config.xml中作如下修改：

deploy:
type: git
repo: git@github.com:gloryyang/gloryyang.github.io.git
branch: master

然后在命令行中执行

hexo d

創建一篇新博客

hexo n “標題”

文章的属性：

layout Layout post或page
title 文章的标题
date 创建日期 文件的创建日期
updated 修改日期 文件的修改日期
comments 是否开启评论 true
tags 标签
categories 分类
permalink url中的名字 文件名

hexo new draft “new draft”

草稿發佈

hexo publish “文章名”

部署、發佈

hexo clean #清除緩存，會刪除public文件夾（不必要）

hexo d -g

或一起執行

hexo clean && hexo g && hexo d
